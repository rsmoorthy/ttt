
## Flow

The app building will flow through the following phases:

- Design stage:  (each task below should be implemented by a separate agent)
  - Create DB schemas  (output in DESIGN.md)
  - Create backend API specs (output in DESIGN.md)
  - Create spec for menus and screens (output in UI.md)
  - Identify modules needed like expressjs, react, tailwind (output in SW.md)

- Coding stage:  (each task below should be implemented by a separate agent)
  - General framework code for back end
  - General framework code for front end, including Home page
  - Write DB schemas in ORM like prisma (if any)
  - Implement individual API (one sub agent for each API)
  - Implement individual screens (one sub agent for each screen)

- Testing stage:  (each task below should be implemented by a separate agent)
  - Test plan: Identify the flow of testing and different aspects to test, Write test cases (output to TESTPLAN.md)
  - Implement test cases (one sub agent for each test case)
